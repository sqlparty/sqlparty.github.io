Dell OpenManage Server Administrator(OMSA)是Dell提供的完整的系统管理系统，功能包括Dell服务器、存储、硬件、网络、日志等的管理。
操作可以通过两种方式实施：

  1. 集中管理- 使用Web界面（GUI）
  2. 命令行-通过操作系统

OMSA介绍


官方介绍页面：http://en.community.dell.com/techcenter/systems-management/w/wiki/1760.openmanage-server-administrator-omsa.aspx

这里使用RedHat 6.2进行安装和使用说明。

shell> wget http://downloads.dell.com/FOLDER01543154M/1/OM-SrvAdmin-Dell-Web-LX-7.3.0-350_A00.RHEL6.x86_64.tar.gz
shell> tar -zxvf OM-SrvAdmin-Dell-Web-LX-7.3.0-350_A00.RHEL6.x86_64.tar.gz
shell> sh linux/supportscripts/srvadmin-install.sh --express

这一步安装了如下的管理组件：
- Server Adminstrator Web Server
- Server Instrumentation
- Storage Management
- Remote Access Controller
