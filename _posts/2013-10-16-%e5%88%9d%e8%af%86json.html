JSON是一种轻量级的数据交换方式，它基于JavaScript的一个子集。其具有标准化，简洁易用的特点，往往作为不同应用间数据交换的首选方式。

JSON的基础结构就是JavaScript中的对象和数组，通过这两种结构的组合、嵌套，可以表示各种复杂的结构。

1）对象：无序的"名称/值对"集合。一个对象以"{"（左括号）开始，"}"（右括号）结束，内含"key:value"对，用逗号","划分，形式如{key:value,key:value,...}

举例：
{ "firstName": "Brett" }
{ "firstName": "Brett", "lastName":"McLaughlin", "email": "aaaa" }

2）数组： 值（value）的有序集合。一个数组以"["（左中括号）开始，"]"（右中括号）结束。值之间使用","（逗号）分隔。

什么是值(value)呢？

对象和数组的重要组成都是值(value)，而值的形式又多种多样，包括 双引号括起来的字符串（string）、数值(number)、true、false、 null、对象或者数组。这样，对象中可以嵌套数组，数组中同样可以存储对象，可以反复嵌套组合，形成了丰富的表达能力。

例如如下这个不同人员的清单，形成了对象内嵌数组，数组再嵌对象：

{ "programmers": [
{ "firstName": "Brett", "lastName":"McLaughlin", "email": "aaaa" },
{ "firstName": "Jason", "lastName":"Hunter", "email": "bbbb" },
{ "firstName": "Elliotte", "lastName":"Harold", "email": "cccc" }
],
"authors": [
{ "firstName": "Isaac", "lastName": "Asimov", "genre": "science fiction" },
{ "firstName": "Tad", "lastName": "Williams", "genre": "fantasy" },
{ "firstName": "Frank", "lastName": "Peretti", "genre": "christian fiction" }
],
"musicians": [
{ "firstName": "Eric", "lastName": "Clapton", "instrument": "guitar" },
{ "firstName": "Sergei", "lastName": "Rachmaninoff", "instrument": "piano" }
] }

与另一种流行的数据交换方式XML而言，JSON的可读性会弱很多，手工编写难度也大一些，但是表达相同的内容，使用的字符量可以少很多。
