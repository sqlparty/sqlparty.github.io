一次安装完JDK后，运行java -version，出现如下报错信息：

Error occurred during initialization of VM
java/lang/ClassNotFoundException: error in opening JAR file /usr/java/jdk1.7.0_45/jre/lib/rt.jar

检查了下，/usr/java/jdk1.7.0_45/jre/lib/rt.jar存在。这是什么原因呢，如何处理？

搜索一番后，有类似的报错信息，有人说是内存不够了，于是清空了一下缓存，发现没有效果，进一步查询，有牛人指出了问题：

JDK的安装文件有损坏！

好吧，再次尝试：

  1. 卸载原安装包
  2. 下载最新完整JDK安装包
  3. 安装最新JDK安装包


在/etc/profile中加入了JAVA_HOME的配置：

export JAVA_HOME=/usr/java/jdk1.7.0_45
export PATH=$JAVA_HOME/bin:$PATH

这样再尝试下：

shell> java -version
java version "1.7.0_45"
Java(TM) SE Runtime Environment (build 1.7.0_45-b18)
Java HotSpot(TM) 64-Bit Server VM (build 24.45-b08, mixed mode)

OK！问题解决！
